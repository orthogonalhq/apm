## What is APM?

APM is the package manager and public registry for agent skills. Discover, install, and share `SKILL.md` packages for AI agents.

## Quick Start

Install the CLI globally:

```bash
npm install -g apm
```

Then install a skill:

```bash
apm install @example/my-skill
```
