## What is OPM?

OPM is the package manager and public registry for agent skills. Discover, install, and share `SKILL.md` packages for AI agents.

## Quick Start

Install the CLI globally:

```bash
npm install -g opm
```

Then install a skill:

```bash
opm install @example/my-skill
```
