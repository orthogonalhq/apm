## Commands

### `apm install`

Install a skill package.

```bash
apm install <package-name>
```

### `apm publish`

Publish a skill package to the registry.

```bash
apm publish
```

### `apm search`

Search for skill packages in the registry.

```bash
apm search <query>
```
